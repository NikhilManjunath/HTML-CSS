Syntax Error vs Semantics Error

Syntax Error

A syntax error is a violation of the syntax/grammar of a programming language. A program will not compile until strict syntax is followed

eg: In python,
print(Hello World) is a syntax error
print('Hello World') is the correct syntax


Semantics Error

Semantics Error is a violation of the meaning of a programming language. These errors are not detected by the compilers. This is because, while the syntax of the code is valid, the logic being used is not.

eg: sum = a-b is a semantic error because while the variable name suggests addition, the two numbers are instead undergoing subtraction
